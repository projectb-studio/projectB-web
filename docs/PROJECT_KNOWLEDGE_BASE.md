## 2. 스타일 가이드 상세

### 델피나 Style Guide
- **팔레트:** Dark Brown #3D2B1F → Brown #6B4C3B → Warm Brown #8B6F5E → Sand #C4A882 → Cream #E8DDD3 → Ivory #F5F0EB → Off-white #FAFAF8
- **액센트:** Gold #B8860B, Sale Red #C75050, Success #2E7D32
- **헤딩 폰트:** Cormorant Garamond (세리프, 고급감)
- **본문 폰트:** Pretendard (산세리프, 가독성)
- **코너:** 둥근 모서리 4~8px
- **상품 사진:** 3:4 세로형
- **버튼:** 브라운 배경 + 아이보리 텍스트, border-radius: 0 (스퀘어 변형 가능)
- **배지:** NEW(아이보리bg), BEST(다크브라운bg), SALE(레드bg)

### 프로젝트 B Style Guide
- **팔레트:** Jet Black #0A0A0A → Rich Black #1A1A1A → Charcoal #333333 → Gray #666666 → Silver #999999 → Light Gray #D4D4D4 → Off-white #F0F0F0 → Snow #FAFAFA
- **액센트:** Sale Red #C75050, Success #2D8F4E
- **헤딩 폰트:** Archivo (기하학적 산세리프)
- **본문 폰트:** Pretendard
- **코너:** 각진 모서리 0px (인더스트리얼)
- **상품 사진:** 1:1 정사각
- **자간:** 매우 넓음 (letter-spacing: 4~6px, uppercase)
- **버튼:** 블랙 배경 + 화이트 텍스트, border-radius: 0
- **배지:** NEW(블랙bg), BEST(오프화이트bg), SALE(레드bg), HANDMADE(아웃라인)
