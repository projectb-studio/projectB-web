# 4. 디자인 시스템 구축 내역

### 컬러 팔레트 (Terracotta + Beige — 11색)
```
메인:     #3B2820(다크) → #FAF7F2(배경) 7단계 그라데이션
액센트:   #B5704F(테라코타), #C75050(세일), #4A7C59(성공)
```

### 타이포그래피
- 영문 헤딩: Archivo (Google Fonts, 400~700)
- 한글 본문: Pretendard Variable (CDN)
- 커스텀 클래스: `.heading-display` (0.25em tracking), `.heading-section` (0.15em tracking)

### UI 컴포넌트 스타일 (CSS 클래스 13개)
- 버튼: `.btn-primary` (테라코타), `.btn-secondary` (아웃라인)
- 뱃지: `.badge-new`, `.badge-best`, `.badge-sale`, `.badge-handmade`
- 기타: `.divider`, `.product-image`, `.no-scrollbar`, `.animate-fade-in`

### Tailwind 커스텀 설정
- 색상 11종, 간격 3종, 폰트 2종, letter-spacing 2종
- border-radius 3단계 (2/4/6px)
- aspect-ratio: product (1:1)
- 플러그인: typography, container-queries

### ★ AI 상세페이지 자동 생성 기능

**개념:** 사장님이 상품 사진만 업로드하면, AI가 자동으로 상세페이지 콘텐츠를 생성해주는 기능.
비개발자인 사장님이 별도 디자인 도구 없이도 전문적인 상품 페이지를 만들 수 있게 해줌.

#### AI가 생성하는 콘텐츠 (현재 Product 스키마 기반)

| 필드 | AI 생성 방식 | 편집 가능 |
|------|-------------|----------|
| `description` | 사진 분석 + 상품명/카테고리 기반 → 감성적 상품 소개문 (2~3문장) | ✅ |
| `details` | 사진에서 크기/형태 추정 + 소재 분석 → 규격 정보 | ✅ (수동 보정 권장) |
| `care` | 소재(tag) 기반 → 관리 방법 자동 매칭 | ✅ |
| `shipping` | 기본 배송 정보 템플릿 자동 채움 | ✅ |
| `tag` | 사진 분석 → 소재 카테고리 추천 (fabric/metal/wood/stone/glass) | ✅ |
| `badge` | 신상품이면 NEW, 할인 설정 시 SALE 자동 | ✅ |
| `slug` | 상품명에서 자동 생성 (slugify 유틸 활용) | ✅ |
| 이미지 최적화 | 업로드 사진 → R2 저장 + 썸네일 자동 생성 (600x600 크롭) | — |
